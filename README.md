# r2Playground

## What is this?

Walkthroughs and other radare2 paraphernalia.

## What's included? 

### Dockerfile

Installs r2 from GitHub, r2ghidra-dec with r2pm, and other essentials.

To get started run 

```bash
docker-compose build
docker-compose up -d
docker exec -it r2playground /bin/bash
```
